# Software

This folder contains the Arduino software implementation used for the Heart Rate Monitoring System.

## Technologies Used

- Arduino IDE
- PulseSensorPlayground Library
- Serial Communication

## Functionalities

- Reads pulse sensor data from analog pin
- Detects heartbeats
- Calculates Beats Per Minute (BPM)
- Displays BPM through Serial Monitor
- Filters noise for improved accuracy
