# Hardware

This folder contains the hardware components and setup diagrams used in the Heart Rate Monitoring System.

## Pulse Sensor

The pulse sensor is used to detect heartbeats by measuring changes in blood flow. It generates analog signals that are processed by the Arduino UNO to calculate Beats Per Minute (BPM).

## Arduino UNO

Arduino UNO serves as the main microcontroller of the system. It receives pulse sensor data, processes the signal, and computes heart rate values in real time.

This diagram illustrates the pin configuration of the Arduino UNO used for connecting the pulse sensor and other peripherals in the system.
