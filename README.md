# Heart Rate Monitoring System

## Overview

A real-time Heart Rate Monitoring System developed using Arduino UNO and Pulse Sensor. The system measures pulse signals, calculates Beats Per Minute (BPM), and displays heart rate data for monitoring purposes.

## Technologies Used

- Arduino UNO
- Pulse Sensor
- Arduino IDE

## Features

- Real-time heart rate monitoring
- BPM calculation
- Pulse signal processing
- Continuous data acquisition

## Applications

- Health monitoring
- Fitness tracking
- Biomedical instrumentation
